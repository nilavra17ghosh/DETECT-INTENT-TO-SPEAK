================================================================================
                     EXPLANATION: Detecting Intent-to-Speak from EEG
================================================================================

1. PROBLEM SETUP
================================================================================

Task: Binary classification — Intent (1) vs No-Intent (0).
Window: −500 ms to 0 ms relative to speech onset.

Positive samples:
   - Real data: 500 ms window immediately before action onset (event 44).
   - Baseline correction uses the first 100 ms of the epoch (code behavior).

Negative samples:
   - Real data: 500 ms rest window immediately after relax onset (event 45).
   - Baseline correction uses the first 100 ms of the epoch (code behavior).

Ground truth:
   - Inner Speech Dataset (ds003626): event markers in the Status channel.
      Event 44 = action onset, Event 45 = relax onset.
   - PhysioNet EEGBCI: cue-locked motor imagery events; used as a proxy task.

2. DATA COLLECTION PLAN
================================================================================

Datasets used in the repo:
   A. Inner Speech Dataset (OpenNeuro ds003626):
       - 10 subjects, 128-channel BioSemi ActiveTwo, 1024 Hz.
       - Overt/inner speech tasks with event markers in Status channel.
       - Used as the primary real dataset for intent-to-speak.

   B. PhysioNet EEGBCI (motor imagery):
       - 1–109 subjects available; typical usage 5 subjects for quick runs.
       - Used as a proxy task for motor preparation (Bereitschaftspotential).
       - Faster to download and run; not true speech, but same SMA/pre-SMA
          readiness mechanism.

   C. Synthetic data:
       - Smoke-test only; outputs marked with [SIMULATION — NOT REAL EEG].
       - Includes BP-like ramps, drift, eye-blinks, and EMG bursts.

Number of subjects / trials:
   - Inner Speech: number of subjects depends on available sub-XX folders.
      The loader balances intent and rest trials per subject.
   - PhysioNet: number of subjects is user-selected in the notebook.

Suitability:
   - Inner Speech is closest to the target task (speech action onset).
   - PhysioNet provides a validated proxy when storage/time is limited.

3. HARDWARE & ELECTRODE CHOICE
================================================================================

Deployment assumption:
   - 4-channel wearable EEG (e.g., OpenBCI Cyton) at 250–256 Hz.
   - Research-grade BioSemi data are downsampled to 256 Hz.

Selected electrodes (2–4):
   FCz (pre-SMA), Cz (SMA), C3 (left M1 mouth), F7 (left IFG/Broca).

Justification:
   - FCz/Cz capture early and late Bereitschaftspotential components.
   - C3 captures motor preparation ERD for articulation.
   - F7 approximates left IFG for phonological planning.

2-channel fallback: FCz + C3.

4. PREPROCESSING
================================================================================

Pipeline for inner speech (real data):
  1. Bandpass 0.1–40 Hz (preserve slow BP, remove DC and high-frequency noise)
  2. Notch 50 Hz (power line removal)
  3. Downsample 1024 → 256 Hz
  4. Epoch extraction: −500 ms to 0 ms relative to action onset (event 44)
  5. Baseline correction: first 100 ms of the epoch (code behavior)
  6. Artifact rejection: ±100 µV threshold (called in notebooks)
  7. Z-score: fit on train split only, apply to val/test (no leakage)

Pipeline for PhysioNet (proxy dataset):
   - Bandpass 8-30 Hz to isolate mu/beta for motor imagery.
   - Cue-locked epochs from MNE with a fixed pre-cue baseline.
   - Notebook default: tmin=0.0, tmax=0.5 (cue to 500 ms post-cue).
      For strict assignment alignment, set tmin=-0.5 and tmax=0.0.

Leakage control:
  - No normalization on the full dataset.
  - Block-wise CV prevents session drift leakage.

5. MODELING
================================================================================

Primary model:
   - EEGNet (compact CNN, ~1.2k params). Single-logit BCE.

Baselines:
   - Riemannian + Logistic Regression (strong classical baseline).
   - ShallowConvNet (band-power baseline).

Optional enhancement:
   - EEGNetWithAttention (lightweight temporal attention; no full Transformer).

Not used as primary:
   - EEG-Conformer (experimental; high overfitting risk with 2–4 channels).

6. TRAINING & LOSS
================================================================================

Loss:
  - BCEWithLogitsLoss with label smoothing (eps = 0.05).

Augmentation (training only):
  - Time shift (±50 ms) for synthetic data.
  - Channel dropout (10%).
  - Mixup (alpha 0.2) in training loop.
  - Real EEG uses mild Gaussian noise instead of circular shift.

Splits:
  - Within-subject: block-wise k-fold CV (blocks from sessions).
  - Cross-subject: LOSO (leave-one-subject-out).

Leakage avoidance:
  - Z-score stats fit on training folds only.
  - No trial-level mixing across blocks.


7. EXPECTED PERFORMANCE (REALISTIC)
================================================================================

Accuracy is modest by design due to: 2–4 electrodes, a 500 ms pre-speech
window, and low SNR of the Bereitschaftspotential.

Typical expectations (from current notebook runs and low-channel EEG literature):
   - Within-subject, EEGNet (4 ch): 0.50–0.60 balanced accuracy
   - Cross-subject (LOSO): 0.48–0.56 balanced accuracy
   - With 30-trial fine-tune: +2–4 pp improvement

If within-subject AUC > 0.90 on real data, audit for leakage:
   1. Epoch ends strictly at speech onset
   2. EMG leakage
   3. Cue-evoked visual potentials
   4. Z-score fit on full dataset

   Scenario                              | Bal. Acc   | AUC-ROC
   --------------------------------------|------------|--------
   Within-subject, EEGNet, 4 ch          | 0.50–0.60  | 0.55–0.62
   Within-subject, 2 ch (FCz+C3)         | 0.45–0.55  | 0.50–0.58
   Cross-subject (LOSO), no fine-tune     | 0.48–0.56  | 0.52–0.60
   Cross-subject + 30-trial calibration   | 0.50–0.58  | 0.54–0.62

LEAKAGE AUDIT: If within-subject AUC > 0.90 on REAL data, investigate:
   1. Auditory cortex contamination — epoch must end strictly at t_onset
   2. EMG leakage — chin-EMG bleeding into frontal channels
   3. Cue-evoked visual potential — verify self-paced trial design
   4. Train/val leakage in normalisation or CV splits

If pipeline produces AUC > 0.90 on SYNTHETIC data, that is the synthetic-
data circularity, not real performance.  The notebook prints a warning.

8. LIMITATIONS
================================================================================

- Low SNR EEG signals and subject variability
- Short 500 ms pre-speech window
- Cross-subject generalization is hard without calibration
- Only 2–4 electrodes limit spatial information

9. EXTRA EVALUATIONS
================================================================================

- Channel ablation: 11 combos of 2–4 electrodes (Riemannian baseline).
- Detection latency: evaluate windows ending at −100 to −500 ms.
- Cross-subject fine-tuning: 30-trial calibration improves LOSO results.
