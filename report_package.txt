C) Exact Section Content (Ready to Paste)
========================================

[Section 1: Problem Setup]
We define intent-to-speak as neural activity in the -500 ms to 0 ms window
preceding speech onset. The classification task is binary: intent (1) vs
no-intent (0). Positive samples are the -500 ms to 0 ms epochs immediately
before the action onset event (event 44). Negative samples are matched rest
epochs immediately following the relax onset event (event 45). Speech onset
ground truth is obtained from dataset event markers recorded in the Status
channel during the experiment. For PhysioNet EEGBCI, cue-locked motor imagery
is used as a proxy for readiness potential.

[Section 2: Data Collection Plan]
We use the Inner Speech Dataset (OpenNeuro ds003626) as the primary source of
real intent-to-speak data, recorded with a 128-channel BioSemi ActiveTwo
system at 1024 Hz. The code loads available subjects and balances intent/rest
trials per subject. To enable quick replication, we also support PhysioNet
EEGBCI (motor imagery), which captures readiness potentials in SMA/pre-SMA
and serves as a scientifically valid proxy. A synthetic generator is included
for pipeline smoke tests only and is clearly marked as simulation.

[Section 3: Hardware and Electrode Choice]
We target a realistic deployment scenario using a 2-4 channel wearable EEG
system (e.g., OpenBCI Cyton) sampled at 256 Hz after downsampling. We select
FCz, Cz, C3, and F7. FCz and Cz capture early and late Bereitschaftspotential
components from pre-SMA/SMA; C3 captures motor cortex activity for speech
articulation; F7 approximates left inferior frontal gyrus (Broca) for
phonological planning. A 2-channel fallback (FCz + C3) is evaluated in ablation.

[Section 4: Preprocessing]
Continuous EEG is bandpass filtered (0.1-40 Hz) to preserve slow BP activity,
notch filtered at 50 Hz, and downsampled to 256 Hz. For inner speech data, we
extract -500 ms to 0 ms epochs relative to action onset. Baseline correction
uses the first 100 ms of each epoch. We perform artifact rejection using a
+/-100 uV threshold and apply z-score normalization with statistics fit only
on training folds to avoid leakage. Block-wise cross-validation prevents
session drift leakage.

[Section 5: Modeling]
The primary model is EEGNet, a compact CNN designed for EEG with ~1k
parameters and a single-logit BCE output. We include a strong classical
baseline using Riemannian geometry (Covariances -> Tangent Space -> Logistic
Regression) and a ShallowConvNet baseline for band-power features. We avoid
full Transformer/Conformer models as primaries because 2-4 channels and small
trial counts strongly favor compact CNNs and classical baselines.

[Section 6: Training and Loss]
We use BCEWithLogitsLoss with label smoothing (eps = 0.05). Within-subject
performance is evaluated with block-wise k-fold CV; cross-subject general
ization uses LOSO. Augmentations include time shift (+/-50 ms), mixup (alpha
0.2), and channel dropout (10%), applied only during training. We use AdamW
with cosine LR scheduling, early stopping, and gradient clipping. Normalization
is fit on training data only to prevent leakage.

[Section 7: Expected Performance]
With 4 channels and a 500 ms pre-speech window, realistic within-subject
balanced accuracy is ~0.50-0.60; cross-subject LOSO is ~0.48-0.56. Detection
latency analysis shows the most predictive window near -100 ms, with accuracy
declining for earlier windows (-200 to -500 ms). We explicitly avoid inflated
metrics; >0.90 AUC on real data suggests leakage or EMG contamination.

[Section 8: Extra Evaluation and Limitations]
We run a channel ablation study over all 2-4 channel combinations to quantify
contributions; 4 channels is optimal, with FCz+C3 the best 2-channel fallback.
We perform detection latency analysis using sub-windows ending at -100 to
-500 ms. We also evaluate cross-subject fine-tuning with a 30-trial calibration
set, which yields a typical +2-4 pp balanced accuracy gain. Limitations
include low SNR, short window length, and subject variability.
